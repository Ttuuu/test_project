<p>There should be some error-checking done here.  For instance, it should return an <a href="http://pubs.opengroup.org/onlinepubs/009696699/basedefs/sys/types.h.html" rel="nofollow"><code>ssize_t</code></a>, allowing it to return a negative value to indicate an error (anything at or above 0 can be seen as a valid count).
