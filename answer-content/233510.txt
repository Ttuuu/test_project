<p>To pass a list of array arguments as individual parameters to a Python function, use the <code>*</code> operator:</p>

<pre><code>TP1 = TensorProduct(*V)
</code></pre>

<p>This is equivalent to <code>TensorProduct(V[0], V[1], V[2],...,V[n])</code></p>
