<p><strong><code>final</code> Keyword</strong></p>

<p>The <code>Game</code>, <code>Tic Tac Toe Board</code>, <code>Tic Tac Toe Board</code>, all have fields which are assigned but never reassigned, </p>

<pre><code>private StringProperty xPlayer = new SimpleStringProperty("X player");
private StringProperty oPlayer = new SimpleStringProperty("O player");
private TicTacticsGame game;
private Button button = new Button();
</code></pre>

<p>They could all use the <code>final</code> keyword. </p>
