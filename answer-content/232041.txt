<p>Much of the code can be eliminated if we use the <a href="https://www.nuget.org/packages/ADO.DataAccessHelper/" rel="nofollow noreferrer">ADO.DataAccessHelper</a> library; its documentation section may help you to know if it meets your requirement.</p>

<p>In brief, it cuts the effort to write and manage connection, command, reader etc.  In addition to that, it uses <code>DbProviderFactory</code>, which makes it more flexible to work with different databases. </p>
