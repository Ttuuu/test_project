<p>Recent versions of .NET have <a href="https://docs.microsoft.com/en-us/dotnet/api/system.double.isnormal" rel="nofollow noreferrer">Double.IsNormal()</a>:</p>

<blockquote>
<pre><code>public static bool IsNormal(double d);
</code></pre>
  
  <p>Returns <code>true</code> if the value is normal; <code>false</code> otherwise.</p>
  
  <p>Applies to</p>
  
  <ul>
  <li>.NET Core 3.1, 3.0, 2.2, 2.1</li>
  <li>.NET Standard 2.1</li>
  </ul>
</blockquote>
