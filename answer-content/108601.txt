<h3>Sorting idiom</h3>

<ul>
<li>Javascript supports sorting pretty much as you'll see in other languages.</li>
<li>Collections have sorting.</li>
<li>Sort functions can take a function delegate for customized sorting.</li>
<li>Typically, compare the desired values and return an integer that means less-than, equal-to, greater-than. Typically -1, 0, 1 respectively. </li>
<li>handling <code>null</code> does not require converting it to a valid value in your set domain. </li>
<li><strong>Read the documentation!</strong></li>
</ul>
