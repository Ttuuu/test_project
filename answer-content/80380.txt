<p>The opening and closing tags don't match in your JSFiddle.</p>

<p>It's not a good idea to use presentational like <code>highlighted</code> for a CSS class name.  It would be better to use <code>current</code> or <code>selected</code>, and let the stylesheet decide how to indicate which item is currently selected â€” whether by highlighting, underlining, or whatever visual hint it chooses.</p>

<p>It would be a lot simpler if you removed the <code>highlighted</code> class from all items, then added it back for just the selected item.</p>
