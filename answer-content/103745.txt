<h2>Design</h2>
<p>There are a lot of missing features that I would expect in unit test framework.</p>
<ul>
<li>Ability to run tests in isolation</li>
<li>Ability to run groups of tests.</li>
<li>Ability to set up state for tests</li>
<li>Ability to tear down state and reset the environment.</li>
<li>Some facility for mocking other classes.</li>
</ul>
