<p>The easiest solution would be to use the <a href="https://pypi.python.org/pypi/aenum" rel="noreferrer"><code>aenum</code></a><sup>1</sup> library, which is drop-in compatible with the stdlib and also has some advanced features not found in the stdlib.  It also works in Python versions 2.7 and 3.3+.</p>
