<p>One possibility is to use a <code>Set</code>. Just add the elements from the array and then get the unique elements from the <code>Set</code>. This algorithm will be <code>O(n)</code> instead of <code>O(n^2)</code> because each element is accessed only once.</p>

<p>Note that you can use <code>Arrays.asList()</code> to simplify the coding, too.</p>
