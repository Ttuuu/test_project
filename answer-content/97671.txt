<p>Please follow PEP8, the coding style guide.</p>

<p>Use boolean expressions directly, no need to have <code>condition == True</code></p>

<p>Avoid global variables. If many methods share data, consider moving the method and the common variables to a class.</p>

<p>Avoid duplicated string literals like "ORANGE".</p>

<p>Avoid duplicated logic, especially when it depends on the operating system like clearing the screen. Encapsulate this in a function.</p>
