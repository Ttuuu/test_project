<p>You could use a <code>while</code> loop instead of this:</p>

<blockquote>
<pre><code>for(; n1&gt;=100; n2--)
</code></pre>
</blockquote>

<p>It would look cleaner like this:</p>

<pre><code>while(n1 &gt;= 100) {
    // ... some code here
    n2--;
}
</code></pre>
