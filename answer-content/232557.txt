<p>This is a <em>parameter object</em> or <em>argument object</em>, which is related to the <em>command pattern</em>.</p>

<p>An example of this in the Java class library is the constraint object <code>java.awt.LayoutManager2</code> (an odd name for a JDK1.00 type).</p>

<p>The Software Engineering StackExchange has the question <a href="https://softwareengineering.stackexchange.com/q/251939/735">Does the pattern of passing in one object instead of many parameters to a constructor have a name?</a> [it does].</p>
