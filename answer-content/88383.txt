<p>Why not count all occurrences of the letters? Also count the string length.</p>

<ol>
<li>If the string length is even and all characters occur even number of times, then it is a palindrome.</li>
<li>If the string length is odd and all characters occur even number of times except for one character that occurs odd number of times, then it is a palindrome.</li>
<li>Otherwise, it is not.</li>
</ol>
