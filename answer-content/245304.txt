<p><strong>Configurations should be stored in a configuration file(.cfg /.ini  /.env) .</strong></p>
<p>You can use <a href="https://pypi.org/project/python-decouple/" rel="nofollow noreferrer">Python Decouple</a> for simple usecase without namespaces. If you want to use namespaces check out the inbuilt <a href="https://docs.python.org/3/library/configparser.html" rel="nofollow noreferrer">ConfigParser</a> module.
This is because if you share your code for review or to some forum to get your errors resolved you might accidentally give your secret information which can allow other users to access your data. This has happened a few times with me when I was sharing error logs and code on IRC chat.</p>
